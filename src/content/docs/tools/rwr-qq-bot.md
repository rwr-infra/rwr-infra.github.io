# RWR QQ Bot

QQ Bot that integrates with OneBOT API for querying server status information in QQ groups.

## Features

- **Server Status**: Query RWR server status directly in QQ
- **OneBOT Integration**: Works with any OneBOT-compatible client
- **Group Commands**: Supports QQ group commands
