# RWRSG

Web interface for creating player trade images, making it easy to share buy/sell item intentions.

## Features

- **Trade Image Generator**: Create visually appealing trade images
- **Shareable**: Easy to share in forums and social media
- **Web-based**: No installation required

## Links

- **Repository**: [github.com/rwr-infra/rwrsg](https://github.com/rwr-infra/rwrsg)

## Use Cases

- Posting item sales in community forums
- Sharing wanted items with other players
- Trading screenshots for promotion
