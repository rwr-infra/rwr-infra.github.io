# VSCode RWR Mod Tool

VSCode extension designed to enhance RWR Modding experience.

## Features

- **AngelScript Support**: Navigate and find references in AngelScript files
- **Code Navigation**: Jump to definitions and find usages
- **XML Validation**: XML file detection and validation for mod configs
- **Modding Productivity**: Tools specifically for RWR mod authors

## Links

- **Repository**: [github.com/rwr-infra/vscode-rwr-mod-tool](https://github.com/rwr-infra/vscode-rwr-mod-tool)
- **VSCode Marketplace**: Search "RWR Mod Tool" in VSCode extensions

## Requirements

- VSCode or compatible editor (Cursor, VSCodium)
- AngelScript files from RWR mods
